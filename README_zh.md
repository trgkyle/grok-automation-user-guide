# 🚀 Grok 自动化 v2.0.4 - Grok.com AI 自动化 [![English](https://img.shields.io/badge/English-blue)](README.md) [![Tiếng Việt](https://img.shields.io/badge/Tiếng%20Việt-green)](README_vi.md)

**Grok 自动化** 是一款强大的生产力工具，旨在增强您在 xAI 的 grok.com 上的创作流程。告别手动逐个输入提示词——实现流程自动化，大规模生成高质量内容。

-----

## ✨ 主要功能

* **🚀 先进的批处理：** 排队数十个或数百个提示词，让扩展程序自动处理提交和生成。
* **🎬 文本转视频自动化：** 从文本描述生成令人惊叹的视频。支持带自定义延迟的批处理。
* **🖼️ 帧转视频：** 使用静态图片并添加自动运动效果，创建动态视频。
* **🧩 素材转视频：** 将多个界面组件或角色图片合成为一段视频动画。
* **🖼️ 文本转图片批量处理：** 同时创建多张图片，支持多种宽高比（16:9、9:16、1:1、2:3、3:2）。
* **🔄 图片转图片：** 使用 AI 结合文本提示词对现有图片进行变换和增强。
* **⚙️ 专业控制套件：**
    * **并发 Prompt：** 同时处理多个提示词以节省时间。
    * **智能延迟：** 设置提示词之间的自定义间隔，以有效管理速率限制。
    * **自动高清 (Upscale) 并下载：** 自动触发高清增强并保存最终结果到您的电脑。
    * **自动更改文件名：** 自动重命名已下载文件以保持整洁。
    * **保存到文件夹：** 为每个项目的下载文件指定自定义子文件夹。
* **🎭 自动添加角色图片：** 根据文件名中的角色名称自动匹配并附加图片到提示词。
* **📊 实时队列管理：** 通过侧边面板（Side Panel）中的可视化状态栏和活跃提示词列表监控您的生成进度。
* **📂 有序的文件管理：** 自动将下载内容按项目分类到文件夹中，保持您的工作空间整洁。
* **🔧 快速修复：** 一键工具，用于恢复视频或图片生成错误。
* **💳 套餐管理：** 登录以查看您的套餐并追踪每日提示词使用情况。
* **🌐 多语言支持：** 提供英文、越南文、中文、韩文、西班牙文和日文版本。

## 📖 使用指南

### 开始使用

1. **访问 Grok.com**
   - 打开 [grok.com/imagine](https://grok.com/imagine)
   - 该扩展程序仅在 Grok 的 "Imagine" 页面上运行。

2. **打开扩展程序**
   - 点击 Chrome 工具栏中的扩展程序图标。建议将其固定以方便访问！

3. **配置批量设置**
   - 在 **控制** 标签页中，您可以设置：
     - **并发 Prompt：** 同时运行的提示词数量。
     - **提示间隔：** 每次提交提示词之间的等待时间。
     - **保存到文件夹：** 下载文件的子文件夹名称。
     - **自动更改文件名：** 开启/关闭自动文件重命名。

4. **选择模式**
   - 从可用的生成模式中选择：**文本转视频**、**帧转视频**、**素材转视频**、**文本转图片** 或 **图片转图片**。

---

### 1. 文本转视频模式

1. 选择 **文本转视频** 模式。
2. 在输入框中输入提示词（每个提示词之间用 **空行** 分隔）。
3. 或者，点击 **上传** 图标从 `.txt` 文件导入提示词列表。
4. 点击 **运行** 开始批量处理。

**示例提示词：**
```
一个充满霓虹灯的未来派赛博朋克城市，雨水倒映着灯光。
摄像机穿过狭窄的小巷。

一个宁静的日本花园，樱花飘落进池塘。
摄像机缓慢放大观察下方游动的锦鲤。
```

---

### 2. 帧转视频模式

1. 选择 **帧转视频** 模式。
2. 点击上传或拖放源图片。
3. 输入提示词（用空行分隔）。每张图片将与每个提示词一起处理。
4. 在 **设置** 中，选择 **图片处理选项**：
   - *仅使用起始帧* — 每个提示词使用一张图片。
   - *使用起始帧和结束帧* — 每个提示词使用两张图片。
5. 点击 **运行**。

---

### 3. 素材转视频模式

1. 选择 **素材转视频** 模式。
2. 上传您想要制作动画的组件图片或角色图片。
3. 输入提示词（用空行分隔）。
4. 可选择开启 **自动添加角色图片**，以自动附加文件名与提示词中角色名称匹配的图片。
5. 在 **设置** 中，设置 **每个提示词最大输入图片数**（1–10）以控制每个提示词使用的图片数量。
6. 点击 **运行**。

---

### 4. 文本转图片模式

1. 选择 **文本转图片** 模式。
2. 为您的图片输入详细描述。
3. 在设置标签页中配置所需的 **宽高比** 和 **图片模型**。
4. 点击 **运行**。

---

### 5. 图片转图片模式

1. 选择 **图片转图片** 模式。
2. 上传您想要变换的源图片。
3. 输入提示词（用空行分隔）。
4. 可选择开启 **自动添加角色图片**，根据文件名自动附加图片。
5. 在 **设置** 中，设置 **每个提示词最大输入图片数**（1–10）。
6. 点击 **运行**。

## ⚙️ 设置配置

访问 **设置** 标签页来自定义您的体验：

| 设置 | 说明 |
| :--- | :--- |
| **默认模式** | 设置默认开启的模式。 |
| **默认宽高比** | 可选择 16:9、9:16、1:1、2:3 或 3:2。 |
| **每个提示词的输出数量（视频）** | 设置每个提示词生成的视频数量（1–4）。 |
| **每个提示词的输出图片数量** | 设置每个提示词生成的图片数量（1–50）。 |
| **并发 Prompt** | 同时处理的提示词数量（1–6）。 |
| **随机延迟** | 处理下一个提示词前的随机等待时间。 |
| **视频模型** | 选择生成模型（如 Grok 3.1 Fast/Quality、Grok 2）。 |
| **图片模型** | 选择用于文本转图片的 AI 模型。 |
| **默认视频选项** | 默认时长：6秒、10秒（需要 Super Grok）、6秒延伸或10秒延伸（需要 Super Grok）。 |
| **默认图片模式选项** | 图片提示词的默认输入模式：新建图片或编辑图片。 |
| **图片处理选项** | 帧转视频用：仅起始帧，或起始帧 + 结束帧。 |
| **每个提示词最大输入图片数** | 素材转视频 / 图片转图片：1–10 张图片。 |
| **最大重试次数** | 失败时的重试次数（1–20）。 |
| **自动下载质量（视频）** | 不下载、480p、480p（高清）或 720p（需要 Super Grok）。 |
| **自动下载质量（图片）** | 不下载或 1k。 |
| **语言** | 在 English、Tiếng Việt、中文、한국어、Español、日本語 之间切换。 |

## 💡 提示与最佳实践

1. **等待时间：** 如果遇到速率限制，请增加设置中的 **随机延迟**。
2. **并发运行：** 建议从 1 个并发提示词开始，根据账号支持情况缓慢增加。
3. **提示词编写：** 尽可能具体！详细的提示词会带来更好的 AI 生成效果。请用清晰的空行分隔多个提示词。
4. **文件组织：** 使用 **保存到文件夹** 字段，将项目下载内容整理到命名子文件夹中。
5. **角色图片：** 将图片文件命名为角色名称（如 `hero.png`、`villain.jpg`），并开启 **自动添加角色图片** 以自动匹配。
6. **视频时长：** 使用 *6秒延伸* 或 *10秒延伸* 选项将提示词串联成一段长视频。最后一个提示词始终使用非延伸时长。
7. **快速修复：** 如果生成卡住或出错，请点击控制标签页中的 **修复错误** 按钮。

## 🔧 故障排除

| 问题 | 解决方案 |
| :--- | :--- |
| **扩展程序未激活** | 确保您位于 [grok.com/imagine](https://grok.com/imagine)。如果需要，请刷新页面。 |
| **生成错误** | Grok 可能正忙。扩展程序将根据您的 **最大重试次数** 设置自动重试。使用 **修复错误** 手动恢复。 |
| **下载无效** | 确保 Chrome 设置中已 **关闭** "在下载前询问每个文件的保存位置"。 |
| **需要登录** | 请确保您已登录您的 Grok.com 账户。 |
| **图片不足** | 在帧转视频模式中，请确保已上传足够数量的图片以匹配提示词数量。 |
